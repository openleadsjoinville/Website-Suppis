## Project Summary
Suppis is a high-end interior design and custom cabinetry firm that integrates the entire process from design to execution. The project is a modern, minimalist landing page for Suppis, focusing on their unique "Suppis Integra" method which connects suppliers and processes for a seamless client experience.

## Tech Stack
- Next.js 15 (App Router)
- React
- TypeScript
- Tailwind CSS
- Framer Motion (Animations)
- Lucide React (Icons)
- Lenis (Smooth Scrolling)
- Shadcn UI (Components)

## Architecture
- `src/app/page.tsx`: Main landing page with multiple sections (Hero, Intro, Integra Diagram, Process, About, Differentials, Testimonials, Showroom).
- `src/components/`: Reusable UI components like `QuotePopup`, `WhatsAppButton`, and Shadcn UI primitives.
- `public/`: Static assets.

## User Preferences
- Clean, sophisticated, and minimalist aesthetic.
- Use of "Suppis Green" (#4A583E) as a primary brand color.
- Interactive elements like the "Suppis Integra" diagram.
- Direct communication via WhatsApp.
- Bilingual/Portuguese content (Portuguese as primary).

## Project Guidelines
- Keep the design high-end and distinctive (avoid "AI slop").
- Use Framer Motion for scroll-driven animations and micro-interactions.
- Maintain a cohesive color palette (Green #4A583E, Off-white #faf9f6, Zinc/Grey accents).
- No comments unless requested.

## Common Patterns
- Section-based layout with clear transitions.
- Responsive design with specific attention to mobile layouts (especially for complex diagrams).
- Use of decorative SVG icons for depth.
- Alternating dark/light blocks for process steps.
